# Laser Puzzle Game
Bounce Lasers around a grid to hit sensors placed around the map

# Controls:  
__Mouse:__  
  **Left Mouse Button**  
    > Click : Rotate Tile CW  
    > Hold  : Drag Tile  
  **Right Mouse Button**  
    > Click : Deletes Tile  
    
__Keyboard:__  
  **Character Keys**  
    > Rotates Tiles CCW / CW  

  **Number Keys**  
    > 1 - Block  
    > 2 - Reflector  
    > 3 - Splitter  
    > 9 - Beam   *Used For Level Editing*  
    > 0 - Sensor *Used For Level Editing*
